# 🌱 Plant Disease Detection using Deep Learning

## 📌 Overview
This project focuses on detecting plant diseases from leaf images using Deep Learning techniques. A Convolutional Neural Network (CNN) model is trained to classify plant leaves into different disease categories.

The goal is to assist farmers and agricultural experts in early disease detection, improving crop yield and reducing losses.

---

## 🚀 Features
- Image-based disease classification
- Deep learning model using CNN
- Data preprocessing and augmentation
- Model training and evaluation
- Prediction on new images

---

## ⚙️ Workflow

### 1. Data Collection
Dataset consists of plant leaf images categorized into multiple disease classes.

---

### 2. Data Preprocessing
- Image resizing
- Normalization
- Data augmentation

---

### 3. Model Building
A Convolutional Neural Network (CNN) is used:
- Convolution Layers
- Pooling Layers
- Fully Connected Layers
- Softmax Output Layer

---

### 4. Model Training
The model is trained using labeled images to learn disease patterns.

---

### 5. Evaluation
- Accuracy
- Loss
- Validation performance

---

### 6. Prediction
The trained model predicts disease class for new leaf images.

---

## 📊 Results
- Achieved good accuracy on validation dataset
